## 📖 About

This repository is my personal journey through **Data Structures and Algorithms**, implemented from scratch in Python. It covers core computer science fundamentals — from basic arrays to complex graph traversals — and includes solutions to **29 LeetCode problems** across various difficulty levels. The goal is to build a deep, practical understanding of DSA concepts while developing strong problem-solving skills.

## 🚀 Getting Started

### Prerequisites
- Python 3.x installed on your machine

### Clone the Repository

```bash
git clone https://github.com/muhammed-anees-pp/DSA.git
cd DSA
```

### Run Any File

```bash
python Array/reversearry.py
python Sorting/merge_sort.py
python Leetcode/20.py
```

No external dependencies are required — all implementations use the Python standard library only.
